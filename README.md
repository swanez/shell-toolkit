# DevOps toolkit

#### A practical DevOps toolkit for infrastructure automation, Linux administration and server operations.

---

#### 1. [`health_check.sh`](scripts/health_check.sh)
- **Description**: Remotely inspects a target server's health over SSH, reporting CPU usage, memory, disk utilization, top processes, and an overall status summary with color-coded output.
- **Usage**: Run the script and enter the hostname/IP of the target server when prompted.
- **Note**: Make sure the script is executable before running: `chmod +x *.sh`
